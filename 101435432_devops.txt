COMP 3104 - DevOps is a course that introduces students to modern software 
development and deployment practices. It covers key concepts such as:

1. Continuous Integration (CI) and Continuous Deployment (CD)
2. Infrastructure as Code (IaC)
3. Automation in software development
4. Monitoring and logging
5. Cloud computing and containerization

This course equips students with essential DevOps skills, enabling them to 
streamline development workflows, improve collaboration between teams, and 
enhance software reliability.
